Block shorthand functions simplify the creation of block inputs. Instead of writing verbose object literals with nested `type` and content properties, you can use these helpers to create blocks in a single function call. Block shorthands are exported as a namespace from `@unbook/core/shorthands`.:

::tabs{.w-full}
:::tabs-item{icon="i-lucide-code" label="Usage"}
```ts
import { blocks } from '@unbook/core/shorthands'
blocks.paragraph('Hello world')
blocks.heading1('Title')
// and more...
```
:::
:::tabs-item{icon="i-lucide-sliders-horizontal" label="Append to Page"}
```ts
const page = await workspace.getPage('page-id')
await page.append([
  blocks.heading1('Welcome'),
  blocks.paragraph('This is a demo of block shorthands.'),
  blocks.divider(),
  blocks.heading2('Features'),
  blocks.bulletedListItem('Easy to read'),
  blocks.bulletedListItem('Type-safe'),
  blocks.bulletedListItem('Consistent API')
  blocks.callout('Pro tip: Use shorthands for cleaner code!', { icon: 'üí°' }, [
    blocks.paragraph('They make block creation much easier.')
    blocks.paragraph('Enjoy using Unbook!')
  ]),
])
```
:::
