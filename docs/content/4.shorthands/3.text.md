Rich text shorthand functions simplify the creation of formatted text content. Instead of constructing complex rich text objects manually, use these helpers for clean, readable code.

## Import shorthands

Text shorthands are exported as a namespace from `@unbook/core`:

```ts [import.ts]
import { text } from '@unbook/core'

// Use via namespace
text.text('Hello world')
text.bold('Important')
text.mentionPage('page-id')
```

Or destructure the functions you need:

```ts [destructure.ts]
const { text: plainText, bold, italic, code } = text
```

## Basic text

### text

Creates a plain text element with optional formatting.

```ts [text.ts]
import { text } from '@unbook/core'

// Simple text
text.text('Hello world')

// Text with link
text.text('Click here', { link: 'https://example.com' })

// Text with multiple annotations
text.text('Important link', {
  bold: true,
  italic: true,
  link: 'https://example.com'
})
```

## Text formatting

### bold

Creates bold text.

```ts [bold.ts]
import { text } from '@unbook/core'

text.bold('Important text')

// Bold with link
text.bold('Click here', 'https://example.com')
```

### italic

Creates italic text.

```ts [italic.ts]
import { text } from '@unbook/core'

text.italic('Emphasized text')
```

### strikethrough

Creates strikethrough text.

```ts [strikethrough.ts]
import { text } from '@unbook/core'

text.strikethrough('Deleted content')
```

### underline

Creates underlined text.

```ts [underline.ts]
import { text } from '@unbook/core'

text.underline('Underlined text')
```

### code

Creates inline code (monospace) text.

```ts [code.ts]
import { text } from '@unbook/core'

text.code('const x = 1')
text.code('npm install')
```

### color

Creates colored text or highlighted text.

```ts [color.ts]
import { text } from '@unbook/core'

// Text colors
text.color('Red text', 'red')
text.color('Blue text', 'blue')
text.color('Gray text', 'gray')

// Background highlights
text.color('Yellow highlight', 'yellow_background')
text.color('Green highlight', 'green_background')
text.color('Pink highlight', 'pink_background')
```

Available colors: `default`, `gray`, `brown`, `orange`, `yellow`, `green`, `blue`, `purple`, `pink`, `red`

Each color also has a `_background` variant for highlighting.

## Equations

### equation

Creates an inline LaTeX equation.

```ts [equation.ts]
import { text } from '@unbook/core'

text.equation('E = mc^2')
text.equation('\\alpha + \\beta = \\gamma')
text.equation('\\frac{a}{b}')
```

## Mentions

### mentionPage

Creates a mention linking to a page.

```ts [mention-page.ts]
import { text } from '@unbook/core'

text.mentionPage('page-uuid-here')
```

### mentionDatabase

Creates a mention linking to a database.

```ts [mention-database.ts]
import { text } from '@unbook/core'

text.mentionDatabase('database-uuid-here')
```

### mentionUser

Creates a mention tagging a user.

```ts [mention-user.ts]
import { text } from '@unbook/core'

// With user object
text.mentionUser({ object: 'user', id: 'user-uuid-here' })
```

### mentionDate

Creates a date mention.

```ts [mention-date.ts]
import { text } from '@unbook/core'

// Single date
text.mentionDate({ start: '2024-01-15' })

// Date range
text.mentionDate({ start: '2024-01-15', end: '2024-01-20' })

// With time
text.mentionDate({
  start: '2024-01-15T09:00:00',
  time_zone: 'America/New_York'
})
```

### mentionLinkPreview

Creates a link preview mention.

```ts [mention-link-preview.ts]
import { text } from '@unbook/core'

text.mentionLinkPreview('https://github.com/shorwood/unbook')
```

## Template mentions

Template mentions are special mentions used in template blocks that resolve dynamically.

### mentionTemplateDate

Creates a template date mention that resolves to the current date/time.

```ts [mention-template-date.ts]
import { text } from '@unbook/core'

// Current timestamp
text.mentionTemplateDate('now')

// Today's date
text.mentionTemplateDate('today')
```

### mentionTemplateUser

Creates a template user mention that resolves to the current user.

```ts [mention-template-user.ts]
import { text } from '@unbook/core'

text.mentionTemplateUser() // Resolves to 'me'
```

## Combine rich text elements

Rich text arrays can combine multiple elements for complex formatting:

```ts [combined.ts]
import { text, blocks } from '@unbook/core'

// Create a paragraph with mixed formatting
const richText = [
  text.text('Hello '),
  text.bold('world'),
  text.text('! This is '),
  text.italic('important'),
  text.text('. See '),
  text.code('README.md'),
  text.text(' for details.')
]

// Use in a paragraph block
blocks.paragraph(richText)
```

## Use with blocks

Text shorthands work seamlessly with block shorthands:

```ts [with-blocks.ts]
import { blocks, text } from '@unbook/core'

// Simple string (auto-converted)
blocks.paragraph('Plain text works too')

// Rich text array
blocks.paragraph([
  text.text('Check out the '),
  text.bold('new features'),
  text.text(' in '),
  text.code('v2.0'),
  text.text('!')
])

// Complex callout
blocks.callout([
  text.color('‚ö†Ô∏è Warning: ', 'orange'),
  text.text('This action is '),
  text.bold('irreversible'),
  text.text('. See '),
  text.mentionPage('docs-page-id'),
  text.text(' for more information.')
], { icon: '‚ö†Ô∏è', color: 'orange_background' })
```

## Build formatted content

Full example combining text and block shorthands:

```ts [full-example.ts]
import { Workspace, blocks, text } from '@unbook/core'
import { createNotionAdapter } from '@unbook/notion'

const adapter = createNotionAdapter({ auth: process.env.NOTION_TOKEN })
const workspace = new Workspace(adapter)

const page = await workspace.getPage('page-id')

await page.append([
  blocks.heading1('Project Update'),
  
  blocks.paragraph([
    text.text('Last updated: '),
    text.mentionDate({ start: new Date().toISOString().split('T')[0] })
  ]),
  
  blocks.divider(),
  
  blocks.heading2('Summary'),
  blocks.paragraph([
    text.text('We have completed the '),
    text.bold('core features'),
    text.text(' and are now in '),
    text.italic('testing phase'),
    text.text('.')
  ]),
  
  blocks.heading2('Highlights'),
  blocks.bulletedListItem([
    text.text('New '),
    text.code('Workspace'),
    text.text(' API')
  ]),
  blocks.bulletedListItem([
    text.color('Performance improvements', 'green'),
    text.text(' ('),
    text.bold('2x faster'),
    text.text(')')
  ]),
  blocks.bulletedListItem([
    text.text('Math support: '),
    text.equation('O(n \\log n)')
  ]),
  
  blocks.callout([
    text.bold('Next steps: '),
    text.text('Review the documentation and provide feedback.')
  ], { icon: 'üìã', color: 'blue_background' })
])
```
